# FIKSI_2026
FIKSI adalah platform pembelajaran berbasis web yang menggabungkan materi dan game quiz interaktif untuk membantu pengguna meningkatkan kesiapan kerja. Dengan pendekatan yang interaktif dan menyenangkan, yang bertujuan untuk meningkatkan soft skill, kemampuan berpikir cepat, serta kepercayaan diri dalam menghadapi dunia kerja.
